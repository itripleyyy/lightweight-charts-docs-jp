Tooltipツールチップ

A tooltip created using an HTML DOM Element which can either track the cursor vertical, or remain fixed to a specific vertical position. The Delta Tooltip is an example of using the Canvas instead of HTML elements to achieve the tooltip. HTML DOM要素を使用して作成されたツールチップで、カーソルの垂直位置を追跡するか、特定の垂直位置に固定されることができます。Delta Tooltipは、HTML要素の代わりにCanvasを使用してツールチップを実現する例です。
