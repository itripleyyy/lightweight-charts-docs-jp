Rectangle Drawing Tool矩形描画ツール

A simple drawing tool for rectangles. Click on the icon in the top left corner to activate drawing mode, and then click on the chart to define the top left and bottom right corners of a rectangle. The color can be changed using the color picker in the top toolbar. 矩形用のシンプルな描画ツールです。左上隅のアイコンをクリックして描画モードを有効にし、その後チャートをクリックして矩形の左上隅と右下隅を定義します。色は上部ツールバーのカラーピッカーを使用して変更できます。
