Getting started始めに

Requirements​要件​

First of all, Lightweight Charts™ is a client-side library. This means that it does not and cannot work on the server-side (i.e. NodeJS), at least out of the box.まず第一に、Lightweight Charts™はクライアントサイドライブラリです。 これは、サーバーサイド（つまりNodeJS）では動作しないし、動作できないことを意味します。少なくとも、標準の状態では。

The code of lightweight-charts package targets the es2016 language specification. Thus, all the browsers you will have to work with should support this language revision (see this compatibility table). If you need to support the previous revisions, you could try to setup a transpilation of the package to the target you need to support in your build system (e.g. by using Babel). If you'll have any issues with that, please raise an issue on github with the details and we'll investigate possible ways to solve it.このlightweight-chartsパッケージのコードは、es2016言語仕様を対象としています。 したがって、作業するすべてのブラウザはこの言語の改訂をサポートしている必要があります（この互換性テーブルを参照してください）。 以前の改訂をサポートする必要がある場合は、ビルドシステムでサポートする必要のあるターゲットにパッケージのトランスパイルを設定することを試みることができます（例：Babelを使用することによって）。 それに関して問題が発生した場合は、詳細を添えてGitHubに問題を提起してください。解決策を調査します。

Installation​インストール​

The first thing you need to do to use lightweight-charts is to install it from npm:lightweight-chartsを使用するために最初に行う必要があるのは、npmからインストールすることです：

npm install --save lightweight-charts

Note that the package is shipped with TypeScript declarations, so you can easily use it within TypeScript code.このパッケージはTypeScriptの宣言と共に提供されているため、TypeScriptコード内で簡単に使用できます。

Creating a chart​チャートの作成​

Once the library has been installed in your repo you're ready to create your first chart.ライブラリがあなたのリポジトリにインストールされたら、最初のチャートを作成する準備が整いました。

First of all, in a file where you would like to create a chart you need to import the library:まず、チャートを作成したいファイルでライブラリをインポートする必要があります：

import { createChart } from 'lightweight-charts';

createChart is the entry-point for creating charts. You can use it to create as many charts as you need:createChartは、チャートを作成するためのエントリーポイントです。必要なだけ多くのチャートを作成するために使用できます：

import { createChart } from 'lightweight-charts';

// ...

// somewhere in your code
const firstChart = createChart(document.getElementById('firstContainer'));
const secondChart = createChart(document.getElementById('secondContainer'));

The result of this function is a IChartApi object, which you need to use to work with a chart instance.この関数の結果は、チャートインスタンスを操作するために必要なIChartApiオブジェクトです。

Creating a series​シリーズを作成する​

Once your chart is created it is ready to display data.チャートが作成されると、データを表示する準備が整います。

The basic primitive to display a data is a series. There are different types of series:データを表示するための基本的なプリミティブはシリーズです。 さまざまな種類のシリーズがあります：

  • Areaエリア
  • Barバー
  • Baselineベースライン
  • Candlestickキャンドルスティック
  • Histogramヒストグラム
  • Lineライン

To create a series with desired type you need to use appropriate method from IChartApi. All of them have the same naming add<type>Series, where <type> is a type of a series you'd like to create:希望するタイプのシリーズを作成するには、IChartApiから適切なメソッドを使用する必要があります。 すべてのメソッドは同じ名前の形式add<type>Seriesを持ち、ここで<type>は作成したいシリーズのタイプです：

import { createChart } from 'lightweight-charts';

const chart = createChart(container);

const areaSeries = chart.addAreaSeries();
const barSeries = chart.addBarSeries();
const baselineSeries = chart.addBaselineSeries();
// ... and so on

Please look at this page for more information about different series types.このページを参照して、さまざまなシリーズタイプに関する詳細情報を確認してください。

Note that a series cannot be transferred from one type to another one since different series types have different data and options types.シリーズは異なるタイプ間で転送することはできません。なぜなら、異なるシリーズタイプは異なるデータとオプションのタイプを持っているからです。

Setting and updating a data​データの設定と更新​

Once your chart and series are created it's time to set data to the series.チャートとシリーズが作成されたら、シリーズにデータを設定する時です。

Note that regardless of the series type, the API calls are the same (the type of the data might be different though).シリーズのタイプに関係なく、API呼び出しは同じです（ただし、データのタイプは異なる場合があります）。

Setting the data to a series​シリーズにデータを設定する​

To set the data (or to replace all data items) to a series you need to use ISeriesApi.setData method:シリーズにデータを設定する（またはすべてのデータ項目を置き換える）には、ISeriesApi.setDataメソッドを使用する必要があります：

const chartOptions = { layout: { textColor: CHART_TEXT_COLOR, background: { type: 'solid', color: CHART_BACKGROUND_COLOR } } };
const chart = createChart(document.getElementById('container'), chartOptions);
const areaSeries = chart.addAreaSeries({
    lineColor: LINE_LINE_COLOR, topColor: AREA_TOP_COLOR,
    bottomColor: AREA_BOTTOM_COLOR,
});
areaSeries.setData([
    { time: '2018-12-22', value: 32.51 },
    { time: '2018-12-23', value: 31.11 },
    { time: '2018-12-24', value: 27.02 },
    { time: '2018-12-25', value: 27.32 },

    { time: '2018-12-28', value: 25.46 },
    { time: '2018-12-29', value: 23.92 },
    { time: '2018-12-30', value: 22.68 },
    { time: '2018-12-31', value: 22.67 },
]);

const candlestickSeries = chart.addCandlestickSeries({
    upColor: BAR_UP_COLOR, downColor: BAR_DOWN_COLOR, borderVisible: false,
    wickUpColor: BAR_UP_COLOR, wickDownColor: BAR_DOWN_COLOR,
});
candlestickSeries.setData([
    { time: '2018-12-22', open: 75.16, high: 82.84, low: 36.16, close: 45.72 },
    { time: '2018-12-23', open: 45.12, high: 53.90, low: 45.12, close: 48.09 },

Updating the data in a series​シリーズ内のデータを更新する​

In a case when your data is updated (e.g. real-time updates) you might want to update the chart as well.データが更新される場合（例えば、リアルタイム更新）、チャートも更新したいかもしれません。

But using ISeriesApi.setData very often might affect the performance and we do not recommend to do this. Also it replaces all series data with the new one, and probably this is not what you're looking for.しかし、ISeriesApi.setDataを頻繁に使用するとパフォーマンスに影響を与える可能性があり、これを推奨しません。 また、すべてのシリーズデータが新しいデータに置き換えられるため、これが求めているものではない可能性があります。

Thus, to update the data you can use a method ISeriesApi.update. It allows you to update the last data item or add a new one much faster without affecting the performance:したがって、データを更新するには、ISeriesApi.updateメソッドを使用できます。 これにより、最後のデータ項目を更新したり、新しい項目を追加したりすることができ、パフォーマンスに影響を与えることなく、より迅速に行えます：

import { createChart } from 'lightweight-charts';

const chart = createChart(container);

const areaSeries = chart.addAreaSeries();
areaSeries.setData([
    // ... other data items
    { time: '2018-12-31', value: 22.67 },
]);

const candlestickSeries = chart.addCandlestickSeries();
candlestickSeries.setData([
    // ... other data items
    { time: '2018-12-31', open: 109.87, high: 114.69, low: 85.66, close: 111.26 },
]);

// sometime later

// update the most recent bar
areaSeries.update({ time: '2018-12-31', value: 25 });
candlestickSeries.update({ time: '2018-12-31', open: 109.87, high: 114.69, low: 85.66, close: 112 });

// creating the new bar
areaSeries.update({ time: '2019-01-01', value: 20 });
candlestickSeries.update({ time: '2019-01-01', open: 112, high: 112, low: 100, close: 101 });
