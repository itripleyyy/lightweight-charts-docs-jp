Chart typesチャートタイプ

Lightweight Charts offers different types of charts to suit various data visualization needs. This article provides an overview of the available chart types and how to create them.Lightweight Chartsは、さまざまなデータ視覚化ニーズに応じた異なるタイプのチャートを提供します。この記事では、利用可能なチャートタイプとその作成方法の概要を説明します。

Standard Time-based Chart​標準の時間ベースチャート​

The standard time-based chart is the most common type, suitable for displaying time series data.標準の時間ベースチャートは、時系列データを表示するのに適した最も一般的なタイプです。

  • Creation method: createChart作成方法: createChart
  • Horizontal scale: Time-based横軸スケール: 時間ベース
  • Use case: General-purpose charting for financial and time series dataユースケース: 財務および時系列データの一般的なチャート作成
import { createChart } from 'lightweight-charts';

const chart = createChart(document.getElementById('container'), options);

This chart type uses time values for the horizontal scale and is ideal for most financial and time series data visualizations.このチャートタイプは、横軸スケールに時間値を使用し、ほとんどの財務および時系列データの視覚化に最適です。

const chartOptions = { layout: { textColor: 'black', background: { type: 'solid', color: 'white' } } };
const chart = createChart(document.getElementById('container'), chartOptions);
const areaSeries = chart.addSeries(AreaSeries, { lineColor: '#2962FF', topColor: '#2962FF', bottomColor: 'rgba(41, 98, 255, 0.28)' });

const data = [{ value: 0, time: 1642425322 }, { value: 8, time: 1642511722 }, { value: 10, time: 1642598122 }, { value: 20, time: 1642684522 }, { value: 3, time: 1642770922 }, { value: 43, time: 1642857322 }, { value: 41, time: 1642943722 }, { value: 43, time: 1643030122 }, { value: 56, time: 1643116522 }, { value: 46, time: 1643202922 }];

areaSeries.setData(data);

chart.timeScale().fitContent();
 

Yield Curve Chart​イールドカーブチャート​

The yield curve chart is specifically designed for displaying yield curves, common in financial analysis.イールドカーブチャートは、金融分析で一般的なイールドカーブを表示するために特別に設計されています。

  • Creation method: createYieldCurveChart作成方法: createYieldCurveChart
  • Horizontal scale: Linearly spaced, defined in monthly time duration units横軸スケール: 線形間隔で、月単位の時間の持続時間で定義されています
  • Key differences:
    • Whitespace is ignored for crosshair and grid linesホワイトスペースは、クロスヘアとグリッドラインでは無視されます
    • Specialized for yield curve representationイールドカーブの表現に特化しています
import { createYieldCurveChart } from 'lightweight-charts';

const chart = createYieldCurveChart(document.getElementById('container'), options);

Use this chart type when you need to visualize yield curves or similar financial data where the horizontal scale represents time durations rather than specific dates.横軸スケールが特定の日付ではなく時間の持続時間を表す場合、イールドカーブや類似の金融データを視覚化する必要があるときにこのチャートタイプを使用してください。

tip

If you want to spread out the beginning of the plot further and don't need a linear time scale, you can enforce a minimum spacing around each point by increasing the minBarSpacing option in the TimeScaleOptions. To prevent the rest of the chart from spreading too wide, adjust the baseResolution to a larger number, such as 12 (months).プロットの開始をさらに広げたい場合で、線形時間スケールが必要ない場合は、minBarSpacingオプションをTimeScaleOptionsで増やすことで、各ポイントの周りに最小の間隔を強制できます。チャートの残りの部分が広がりすぎないように、baseResolutionをより大きな数（例えば12（月））に調整してください。

const chartOptions = {
    layout: { textColor: 'black', background: { type: 'solid', color: 'white' } },
    yieldCurve: { baseResolution: 1, minimumTimeRange: 10, startTimeRange: 3 },
    handleScroll: false, handleScale: false,
};

const chart = createYieldCurveChart(document.getElementById('container'), chartOptions);
const lineSeries = chart.addSeries(LineSeries, { color: '#2962FF' });

const curve = [{ time: 1, value: 5.378 }, { time: 2, value: 5.372 }, { time: 3, value: 5.271 }, { time: 6, value: 5.094 }, { time: 12, value: 4.739 }, { time: 24, value: 4.237 }, { time: 36, value: 4.036 }, { time: 60, value: 3.887 }, { time: 84, value: 3.921 }, { time: 120, value: 4.007 }, { time: 240, value: 4.366 }, { time: 360, value: 4.290 }];

lineSeries.setData(curve);

chart.timeScale().fitContent();
 

Options Chart (Price-based)​オプションチャート（価格ベース）​

The options chart is a specialized type that uses price values on the horizontal scale instead of time.オプションチャートは、時間の代わりに横軸スケールに価格値を使用する特化型のチャートです。

  • Creation method: createOptionsChart作成方法: createOptionsChart
  • Horizontal scale: Price-based (numeric)横軸スケール: 価格ベース（数値）
  • Use case: Visualizing option chains, price distributions, or any data where price is the primary x-axis metricユースケース: オプションチェーン、価格分布、または価格が主なx軸メトリックであるデータの視覚化
import { createOptionsChart } from 'lightweight-charts';

const chart = createOptionsChart(document.getElementById('container'), options);

This chart type is particularly useful for financial instruments like options, where the price is a more relevant x-axis metric than time.このチャートタイプは、オプションのような金融商品に特に有用で、価格が時間よりも関連性の高いx軸メトリックです。

const chartOptions = {
    layout: { textColor: 'black', background: { type: 'solid', color: 'white' } },
};

const chart = createOptionsChart(document.getElementById('container'), chartOptions);
const lineSeries = chart.addSeries(LineSeries, { color: '#2962FF' });

const data = [];
for (let i = 0; i < 1000; i++) {
    data.push({
        time: i * 0.25,
        value: Math.sin(i / 100) + i / 500,
    });
}

lineSeries.setData(data);

chart.timeScale().fitContent();
 

Custom Horizontal Scale Chart​カスタム横軸スケールチャート​

For advanced use cases, Lightweight Charts allows creating charts with custom horizontal scale behavior.高度なユースケースのために、Lightweight Chartsはカスタム横軸スケール動作を持つチャートの作成を可能にします。

  • Creation method: createChartEx作成方法: createChartEx
  • Horizontal scale: Custom-defined横軸スケール: カスタム定義
  • Use case: Specialized charting needs with non-standard horizontal scalesユースケース: 非標準の横軸スケールを持つ特化したチャートニーズ
import { createChartEx, defaultHorzScaleBehavior } from 'lightweight-charts';

const customBehavior = new (defaultHorzScaleBehavior())();
// Customize the behavior as needed

const chart = createChartEx(document.getElementById('container'), customBehavior, options);

This method provides the flexibility to define custom horizontal scale behavior, allowing for unique and specialized chart types.この方法は、カスタム横軸スケール動作を定義する柔軟性を提供し、ユニークで特化したチャートタイプを可能にします。

Choosing the Right Chart Type​適切なチャートタイプの選択​

  • Use createChart for most standard time-based charting needs.createChartを使用して、ほとんどの標準的な時間ベースのチャート作成ニーズに対応します。
  • Choose createYieldCurveChart when working specifically with yield curves or similar financial data.利回り曲線や類似の金融データを扱う場合は、createYieldCurveChartを選択してください。
  • Opt for createOptionsChart when you need to visualize data with price as the primary horizontal axis, such as option chains.オプションチェーンのように、価格を主な横軸としてデータを視覚化する必要がある場合は、createOptionsChartを選択してください。
  • Use createChartEx when you need a custom horizontal scale behavior that differs from the standard time-based or price-based scales.標準の時間ベースまたは価格ベースのスケールとは異なるカスタム横スケールの動作が必要な場合は、createChartExを使用してください。

Each chart type provides specific functionality and is optimized for different use cases. Consider your data structure and visualization requirements when selecting the appropriate chart type for your application.各チャートタイプは特定の機能を提供し、異なるユースケースに最適化されています。アプリケーションに適したチャートタイプを選択する際には、データ構造と視覚化要件を考慮してください。
