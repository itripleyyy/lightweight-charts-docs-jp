Custom Series Typesカスタムシリーズタイプ

Custom series allow developers to create new types of series with their own data structures, and rendering logic (implemented using CanvasRenderingContext2D methods). These custom series extend the current capabilities of our built-in series, providing a consistent API which mirrors the built-in chart types.カスタムシリーズは、開発者が独自のデータ構造とレンダリングロジックを持つ新しいタイプのシリーズを作成できるようにします（CanvasRenderingContext2Dメソッドを使用して実装）。これらのカスタムシリーズは、組み込みのシリーズの現在の機能を拡張し、組み込みのチャートタイプを反映する一貫したAPIを提供します。

note

These series are expected to have a uniform width for each data point, which ensures that the chart maintains a consistent look and feel across all series types. The only restriction on the data structure is that it should extend the CustomData interface (have a valid time property for each data point).これらのシリーズは、各データポイントの幅が均一であることが期待されており、これによりチャートはすべてのシリーズタイプで一貫した外観と感触を維持します。データ構造に関する唯一の制約は、CustomDataインターフェースを拡張する必要があることです（各データポイントに有効な時間プロパティを持つこと）。

Defining a Custom Series​カスタムシリーズの定義​

A custom series should implement the ICustomSeriesPaneView interface. The interface defines the basic functionality and structure required for creating a custom series view.カスタムシリーズは、ICustomSeriesPaneViewインターフェースを実装する必要があります。このインターフェースは、カスタムシリーズビューを作成するために必要な基本的な機能と構造を定義します。

It includes the following methods and properties:以下のメソッドとプロパティが含まれています：

Renderer​レンダラー​

  • ICustomSeriesPaneView property: rendererICustomSeriesPaneView プロパティ: renderer

This method should return a renderer which implements the ICustomSeriesPaneRenderer interface and is used to draw the series data on the main chart pane.このメソッドは、ICustomSeriesPaneRenderer インターフェースを実装するレンダラーを返す必要があり、メインチャートペインにシリーズデータを描画するために使用されます。

The draw method of the renderer is evoked whenever the chart needs to draw the series.draw メソッドは、チャートがシリーズを描画する必要があるときにレンダラーによって呼び出されます。

The PriceToCoordinateConverter provided as the 2nd argument to the draw method is a convenience function for changing prices into vertical coordinate values. It is provided since the series' original data will most likely be defined in price values, and the renderer needs to draw with coordinates. The values returned by the converter will be defined in mediaSize (unscaled by devicePixelRatio).PriceToCoordinateConverter は、draw メソッドの2番目の引数として提供され、価格を垂直座標値に変換するための便利な関数です。シリーズの元のデータはおそらく価格値で定義されるため、レンダラーは座標で描画する必要があります。コンバーターによって返される値は、mediaSize で定義され（devicePixelRatio でスケーリングされていません）。

tip

CanvasRenderingTarget2D provided within the draw function is explained in more detail on the Canvas Rendering Target page.CanvasRenderingTarget2D は、draw 関数内で提供され、Canvas Rendering Target ページで詳細に説明されています。

Update​更新​

  • ICustomSeriesPaneView property: updateICustomSeriesPaneView プロパティ: update

This method will be called with the latest data for the renderer to use during the next paint.このメソッドは、次回の描画中にレンダラーが使用するための最新データで呼び出されます。

The update method is evoked with two parameters: data (discussed below), and seriesOptions. seriesOptions is a reference to the currently applied options for the seriesupdate メソッドは、data（以下で説明）と seriesOptions の2つのパラメータで呼び出されます。seriesOptions は、現在適用されているシリーズのオプションへの参照です。

The PaneRendererCustomData interface provides the data that can be used within the renderer for drawing the series data. It includes the following properties:PaneRendererCustomData インターフェースは、シリーズデータを描画するためにレンダラー内で使用できるデータを提供します。以下のプロパティが含まれています:

  • bars: List of all the series' items and their x coordinates. See CustomBarItemData for more detailsbars: シリーズのすべてのアイテムとそのx座標のリスト。詳細については、CustomBarItemDataを参照してください。
  • barSpacing: Spacing between consecutive bars.barSpacing: 連続するバー間の間隔。
  • visibleRange: The current visible range of items on the chart.visibleRange: チャート上の現在の表示範囲。

Price Value Builder​

  • ICustomSeriesPaneView property: priceValueBuilderICustomSeriesPaneViewプロパティ: priceValueBuilder

A function for interpreting the custom series data and returning an array of numbers representing the prices values for the item, specifically the equivalent highest, lowest, and current price values for the data item.カスタムシリーズデータを解釈し、アイテムの価格値を表す数値の配列を返す関数。具体的には、データアイテムの同等の最高、最低、現在の価格値を返します。

These price values are used by the chart to determine the auto-scaling (to ensure the items are in view) and the crosshair and price line positions. The largest and smallest values in the array will be used to specify the visible range of the painted item, and the last value will be used for the crosshair and price line position.これらの価格値は、アイテムが表示されるように自動スケーリングを決定するためにチャートによって使用され、クロスヘアと価格ラインの位置を決定します。配列内の最大値と最小値は、描画されたアイテムの表示範囲を指定するために使用され、最後の値はクロスヘアと価格ラインの位置に使用されます。

Whitespace​

  • ICustomSeriesPaneView property: isWhitespaceICustomSeriesPaneViewプロパティ: isWhitespace

A function used by the library to determine which data points provided by the user should be considered Whitespace. The method should return true when the data point is Whitespace. Data points which are whitespace data won't be provided to the renderer, or the priceValueBuilder.ライブラリによって使用される関数で、ユーザーが提供したデータポイントの中でどれがWhitespaceと見なされるべきかを判断します。このメソッドは、データポイントがWhitespaceである場合にtrueを返す必要があります。Whitespaceデータのデータポイントは、レンダラーやpriceValueBuilderに提供されません。

Default Options​デフォルトオプション​

  • ICustomSeriesPaneView property: defaultOptionsICustomSeriesPaneView プロパティ: defaultOptions

The default options to be used for the series. The user can override these values using the options argument in addCustomSeries, or via the applyOptions method on the ISeriesAPI.シリーズに使用されるデフォルトオプションです。ユーザーは、 addCustomSeries のオプション引数を使用するか、 applyOptions メソッドを介してこれらの値を上書きできます。 ISeriesAPI。

Destroy​破棄​

  • ICustomSeriesPaneView property: destroyICustomSeriesPaneView プロパティ: destroy

This method will be evoked when the series has been removed from the chart. This method should be used to clean up any objects, references, and other items that could potentially cause memory leaks.このメソッドは、シリーズがチャートから削除されたときに呼び出されます。この メソッドは、メモリリークを引き起こす可能性のあるオブジェクト、参照、およびその他のアイテムをクリーンアップするために使用する必要があります。

This method should contain all the necessary code to clean up the object before it is removed from memory. This includes removing any event listeners or timers that are attached to the object, removing any references to other objects, and resetting any values or properties that were modified during the lifetime of the object.このメソッドには、オブジェクトがメモリから削除される前にクリーンアップするために必要なすべてのコードが含まれている必要があります。これには、オブジェクトに関連付けられたイベントリスナーやタイマーの削除、他のオブジェクトへの参照の削除、およびオブジェクトのライフサイクル中に変更された値やプロパティのリセットが含まれます。
