getVisibleRange()​

getVisibleRange(): IRange<HorzScaleItem>

Returns current visible time range of the chart.チャートの現在の表示時間範囲を返します。

Note that this method cannot extrapolate time and will use the only currently existent data. To get complete information about current visible range, please use getVisibleLogicalRange and ISeriesApi.barsInLogicalRange.このメソッドは時間を外挿することができず、現在存在するデータのみを使用することに注意してください。 現在の表示可能範囲に関する完全な情報を取得するには、getVisibleLogicalRangeおよびISeriesApi.barsInLogicalRangeを使用してください。

Returns​戻り値​

IRange<HorzScaleItem>

Visible range or null if the chart has no data at all.可視範囲、またはデータが全くない場合はnull。

setVisibleRange()​

setVisibleRange(range): void

Sets visible range of data.データの表示可能範囲を設定します。

Note that this method cannot extrapolate time and will use the only currently existent data. Thus, for example, if currently a chart doesn't have data prior 2018-01-01 date and you set visible range with from date 2016-01-01, it will be automatically adjusted to 2018-01-01 (and the same for to date).このメソッドは時間を外挿することができず、現在存在するデータのみを使用することに注意してください。 したがって、例えば、現在チャートに2018-01-01の日付以前のデータがない場合、from日付2016-01-01で表示可能範囲を設定すると、自動的に2018-01-01に調整されます（to日付についても同様です）。

But if you can approximate indexes on your own - you could use setVisibleLogicalRange instead.しかし、自分でインデックスを近似できる場合は、setVisibleLogicalRangeを代わりに使用することができます。

Parameters​パラメータ​

• range: IRange<HorzScaleItem>

Target visible range of data.データの対象表示可能範囲。

Returns​戻り値​

void

Example​例​

chart.timeScale().setVisibleRange({
    from: (new Date(Date.UTC(2018, 0, 1, 0, 0, 0, 0))).getTime() / 1000,
    to: (new Date(Date.UTC(2018, 1, 1, 0, 0, 0, 0))).getTime() / 1000,
});

subscribeVisibleTimeRangeChange()​

subscribeVisibleTimeRangeChange(handler): void

Subscribe to the visible time range change events.可視の時間範囲の変更イベントに登録します。

The argument passed to the handler function is an object with from and to properties of type Time, or null if there is no visible data.ハンドラ関数に渡される引数は、fromおよびtoプロパティを持つオブジェクトで、型はTime、または可視データがない場合はnullです。

Parameters​パラメータ​

• handler: TimeRangeChangeEventHandler<HorzScaleItem>

Handler (function) to be called when the visible indexes change.可視インデックスが変更されたときに呼び出されるハンドラー（関数）。

Returns​返します​

void

Example​例​

function myVisibleTimeRangeChangeHandler(newVisibleTimeRange) {
    if (newVisibleTimeRange === null) {
        // handle null
    }

    // handle new logical range
}

chart.timeScale().subscribeVisibleTimeRangeChange(myVisibleTimeRangeChangeHandler);
