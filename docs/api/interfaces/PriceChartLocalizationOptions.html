Version: 5.0

Interface: PriceChartLocalizationOptionsインターフェース: PriceChartLocalizationOptions

Extends LocalizationOptions for price-based charts. Includes settings specific to formatting price values on the horizontal scale.価格ベースのチャート用にLocalizationOptionsを拡張します。 水平スケール上の価格値のフォーマットに特有の設定を含みます。

Extends​拡張​

  • LocalizationOptions <HorzScalePriceItem>

Properties​プロパティ​

timeFormatter?​

optional timeFormatter: TimeFormatterFn<number>

Override formatting of the time scale crosshair label.時間スケールのクロスヘアラベルのフォーマットをオーバーライドします。

Default Value​デフォルト値​

undefined

Inherited from​から継承​

LocalizationOptions . timeFormatter

dateFormat​

dateFormat: string

Date formatting string.日付フォーマットの文字列。

Can contain yyyy, yy, MMMM, MMM, MM and dd literals which will be replaced with corresponding date's value.yyyy, yy, MMMM, MMM, MM および dd のリテラルを含むことができ、これらは対応する日付の値に置き換えられます。

Ignored if timeFormatter has been specified.timeFormatter が指定されている場合は無視されます。

Default Value​デフォルト値​

'dd MMM \'yy'

Inherited from​から継承​

LocalizationOptions . dateFormat

locale​ロケール​

locale: stringロケール: string

Current locale used to format dates. Uses the browser's language settings by default.日付をフォーマットするために使用される現在のロケール。デフォルトではブラウザの言語設定を使用します。

See​

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl#Locale_identification_and_negotiation

Default Value​デフォルト値​

navigator.language

Inherited from​継承元​

LocalizationOptions . locale

priceFormatter?​

optional priceFormatter: PriceFormatterFn

Override formatting of the price scale tick marks, labels and crosshair labels. Can be used for cases that can't be covered with built-in price formats.価格スケールのティックマーク、ラベル、クロスヘアラベルのフォーマットをオーバーライドします。組み込みの価格フォーマットではカバーできないケースに使用できます。

See​参照​

PriceFormatCustom

Default Value​デフォルト値​

undefined

Inherited from​継承元​

LocalizationOptions . priceFormatter

tickmarksPriceFormatter?​

optional tickmarksPriceFormatter: TickmarksPriceFormatterFn

Overrides the formatting of price scale tick marks. Use this to define formatting rules based on all provided price values.価格スケールのティックマークのフォーマットをオーバーライドします。提供されたすべての価格値に基づいてフォーマットルールを定義するために使用します。

Default Value​デフォルト値​

undefined

optional percentageFormatter: PercentageFormatterFn

Overrides the formatting of percentage scale tick marks.パーセンテージスケールの目盛りのフォーマットを上書きします。

tickmarksPercentageFormatter?​

optional tickmarksPercentageFormatter: TickmarksPercentageFormatterFn

Override formatting of the percentage scale tick marks. Can be used if formatting should be adjusted based on all the values being formattedパーセンテージスケールの目盛りのフォーマットを上書きします。フォーマットする値に基づいて調整が必要な場合に使用できます。

Default Value​デフォルト値​

undefined

precision​

precision: number精度: number

The number of decimal places to display for price values on the horizontal scale.横軸の価格値に表示する小数点以下の桁数。
