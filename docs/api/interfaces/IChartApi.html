resize()​

resize(width, height, forceRepaint?): void

Sets fixed size of the chart. By default chart takes up 100% of its container.チャートの固定サイズを設定します。デフォルトでは、チャートはそのコンテナの100%を占めます。

If chart has the autoSize option enabled, and the ResizeObserver is available then the width and height values will be ignored.チャートに autoSize オプションが有効で、ResizeObserver が利用可能な場合、幅と高さの値は無視されます。

Parameters​パラメータ​

• width: number

Target width of the chart.チャートの目標幅。

• height: number

Target height of the chart.チャートの目標高さ。

• forceRepaint?: boolean

True to initiate resize immediately. One could need this to get screenshot immediately after resize.リサイズを直ちに開始するためにtrue。リサイズ後にすぐにスクリーンショットを取得する必要がある場合に使用します。

Returns​戻り値​

void

addCustomSeries()​

addCustomSeries<TData, TOptions, TPartialOptions>(customPaneView, customOptions?, paneIndex?): ISeriesApi<"Custom", Time, TData | WhitespaceData <Time>, TOptions, TPartialOptions>

Creates a custom series with specified parameters.指定されたパラメータでカスタムシリーズを作成します。

A custom series is a generic series which can be extended with a custom renderer to implement chart types which the library doesn't support by default.カスタムシリーズは、ライブラリがデフォルトでサポートしていないチャートタイプを実装するためにカスタムレンダラーで拡張できる汎用シリーズです。

Type parameters​型パラメータ​

• TData extends CustomData <Time>• TData は CustomData <Time> を拡張します。

• TOptions extends CustomSeriesOptions• TOptions は CustomSeriesOptions を拡張します。

• TPartialOptions extends DeepPartial<TOptions & SeriesOptionsCommon> = DeepPartial<TOptions & SeriesOptionsCommon>• TPartialOptions は DeepPartial<TOptions & SeriesOptionsCommon> = DeepPartial<TOptions & SeriesOptionsCommon> を拡張します。

Parameters​パラメータ​

• customPaneView: ICustomSeriesPaneView <Time, TData, TOptions>

A custom series pane view which implements the custom renderer.カスタムレンダラーを実装するカスタムシリーズペインビュー。

• customOptions?: DeepPartial<TOptions & SeriesOptionsCommon>

Customization parameters of the series being created.作成中のシリーズのカスタマイズパラメータ。

const series = chart.addCustomSeries(myCustomPaneView);

• paneIndex?: number

Returns​戻り値​

ISeriesApi<"Custom", Time, TData | WhitespaceData <Time>, TOptions, TPartialOptions>

Inherited from​継承元​

IChartApiBase . addCustomSeries

setCrosshairPosition()​

setCrosshairPosition(price, horizontalPosition, seriesApi): void

Set the crosshair position within the chart.チャート内のクロスヘアの位置を設定します。

Usually the crosshair position is set automatically by the user's actions. However in some cases you may want to set it explicitly.通常、クロスヘアの位置はユーザーの操作によって自動的に設定されます。しかし、場合によっては明示的に設定したいことがあります。

For example if you want to synchronise the crosshairs of two separate charts.例えば、2つの別々のチャートのクロスヘアを同期させたい場合です。

Parameters​パラメータ​

• price: number

The price (vertical coordinate) of the new crosshair position.新しいクロスヘア位置の価格（縦座標）。

• horizontalPosition: Time

The horizontal coordinate (time by default) of the new crosshair position.新しいクロスヘア位置の水平座標（デフォルトは時間）。

• seriesApi: ISeriesApi<keyof SeriesOptionsMap, Time, CustomData <Time> | WhitespaceData <Time> | AreaData <Time> | BarData <Time> | CandlestickData <Time> | BaselineData <Time> | LineData <Time> | HistogramData <Time> | CustomSeriesWhitespaceData <Time>, CustomSeriesOptions | AreaSeriesOptions | BarSeriesOptions | CandlestickSeriesOptions | BaselineSeriesOptions | LineSeriesOptions | HistogramSeriesOptions, DeepPartial <AreaStyleOptions & SeriesOptionsCommon> | DeepPartial <BarStyleOptions & SeriesOptionsCommon> | DeepPartial <CandlestickStyleOptions & SeriesOptionsCommon> | DeepPartial <BaselineStyleOptions & SeriesOptionsCommon> | DeepPartial <LineStyleOptions & SeriesOptionsCommon> | DeepPartial <HistogramStyleOptions & SeriesOptionsCommon> | DeepPartial <CustomStyleOptions & SeriesOptionsCommon>>

Returns​返します​

void
