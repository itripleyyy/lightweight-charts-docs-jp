Version: 5.0

Interface: HorzScaleOptionsインターフェース: HorzScaleOptions

Options for the time scale; the horizontal scale at the bottom of the chart that displays the time of data.時間スケールのオプション; チャートの下部に表示されるデータの時間を示す水平スケール。

Extended by​​によって拡張されます

  • TimeScaleOptions

Properties​プロパティ​

rightOffset​

rightOffset: number

The margin space in bars from the right side of the chart.チャートの右側からのバーのマージンスペース。

Default Value​デフォルト値​

0

rightOffsetPixels?​

optional rightOffsetPixels: number

The margin space in pixels from the right side of the chart. This option has priority over rightOffset.チャートの右側からのピクセル単位のマージンスペースです。 このオプションは rightOffset よりも優先されます。

Default Value​デフォルト値​

undefined

barSpacing​

barSpacing: number

The space between bars in pixels.バー間のスペース（ピクセル単位）。

Default Value​デフォルト値​

6

minBarSpacing​

minBarSpacing: number

The minimum space between bars in pixels.バー間の最小スペース（ピクセル単位）。

Default Value​デフォルト値​

0.5

maxBarSpacing​

maxBarSpacing: number

The maximum space between bars in pixels.バー間の最大スペース（ピクセル単位）です。

Has no effect if value is set to 0.値が 0 に設定されている場合は効果がありません。

Default Value​デフォルト値​

0

fixLeftEdge​

fixLeftEdge: boolean

Prevent scrolling to the left of the first bar.最初のバーの左側へのスクロールを防ぎます。

Default Value​デフォルト値​

false

fixRightEdge​

fixRightEdge: boolean

Prevent scrolling to the right of the most recent bar.最新のバーの右側へのスクロールを防ぎます。

Default Value​デフォルト値​

false

lockVisibleTimeRangeOnResize​

lockVisibleTimeRangeOnResize: boolean

Prevent changing the visible time range during chart resizing.チャートのサイズ変更中に表示される時間範囲が変更されるのを防ぎます。

Default Value​デフォルト値​

false

rightBarStaysOnScroll​

rightBarStaysOnScroll: boolean

Prevent the hovered bar from moving when scrolling.スクロール時にホバーされたバーが動くのを防ぎます。

Default Value​デフォルト値​

false

borderVisible​

borderVisible: boolean

Show the time scale border.時間スケールの境界を表示します。

Default Value​デフォルト値​

true

borderColor​

borderColor: string

The time scale border color.時間スケールの境界の色。

Default Value​デフォルト値​

'#2B2B43'

visible​

visible: boolean

Show the time scale.時間スケールを表示します。

Default Value​デフォルト値​

true

timeVisible​

timeVisible: boolean

Show the time, not just the date, in the time scale and vertical crosshair label.時間スケールと垂直クロスヘアラベルに日付だけでなく時間も表示します。

Default Value​デフォルト値​

false

secondsVisible​

secondsVisible: boolean

Show seconds in the time scale and vertical crosshair label in hh:mm:ss format for intraday data.インターデイデータのために、hh:mm:ss形式で時間スケールと垂直クロスヘアラベルに秒を表示します。

Default Value​デフォルト値​

true

shiftVisibleRangeOnNewBar​

shiftVisibleRangeOnNewBar: boolean

Shift the visible range to the right (into the future) by the number of new bars when new data is added.新しいデータが追加されたときに、新しいバーの数だけ可視範囲を右（未来）にシフトします。

Note that this only applies when the last bar is visible.これは最後のバーが表示されている場合にのみ適用されることに注意してください。

Default Value​デフォルト値​

true

allowShiftVisibleRangeOnWhitespaceReplacement​allowShiftVisibleRangeOnWhitespaceReplacement​

allowShiftVisibleRangeOnWhitespaceReplacement: boolean

Allow the visible range to be shifted to the right when a new bar is added which is replacing an existing whitespace time point on the chart.新しいバーが追加されるときに、チャート上の既存のホワイトスペースの時間ポイントを置き換える場合、可視範囲を右にシフトできるようにします。

Note that this only applies when the last bar is visible & shiftVisibleRangeOnNewBar is enabled.これは、最後のバーが可視であり、かつ shiftVisibleRangeOnNewBar が有効な場合にのみ適用されることに注意してください。

Default Value​デフォルト値​

false

ticksVisible​

ticksVisible: boolean

Draw small vertical line on time axis labels.時間軸のラベルに小さな縦線を描画します。

Default Value​デフォルト値​

false

tickMarkMaxCharacterLength?​tickMarkMaxCharacterLength?​

optional tickMarkMaxCharacterLength: number

Maximum tick mark label length. Used to override the default 8 character maximum length.最大ティックマークラベルの長さ。デフォルトの8文字の最大長を上書きするために使用されます。

Default Value​デフォルト値​

undefined

uniformDistribution​uniformDistribution​

uniformDistribution: boolean

Changes horizontal scale marks generation. With this flag equal to true, marks of the same weight are either all drawn or none are drawn at all.水平スケールマークの生成を変更します。 このフラグが true に設定されている場合、同じ重さのマークはすべて描画されるか、まったく描画されないかのいずれかです。

minimumHeight​

minimumHeight: number

Define a minimum height for the time scale. Note: This value will be exceeded if the time scale needs more space to display it's contents.時間スケールの最小高さを定義します。 注意: この値は、時間スケールが内容を表示するためにより多くのスペースを必要とする場合に超えられます。

Setting a minimum height could be useful for ensuring that multiple charts positioned in a horizontal stack each have an identical time scale height, or for plugins which require a bit more space within the time scale pane.最小高さを設定することは、水平スタックに配置された複数のチャートがそれぞれ同一の時間スケールの高さを持つことを保証するためや、時間スケールペイン内で少し多くのスペースを必要とするプラグインのために便利です。

Default Value​デフォルト値​

0

allowBoldLabels​

allowBoldLabels: boolean

Allow major time scale labels to be rendered in a bolder font weight.主要な時間スケールラベルを太字のフォントウェイトでレンダリングできるようにします。

Default Value​デフォルト値​

true

ignoreWhitespaceIndices​

ignoreWhitespaceIndices: boolean

Ignore time scale points containing only whitespace (for all series) when drawing grid lines, tick marks, and snapping the crosshair to time scale points.グリッドライン、目盛り、およびクロスヘアを時間スケールポイントにスナップする際に、空白のみを含む時間スケールポイントを無視します（すべての系列に対して）。

For the yield curve chart type it defaults to true.イールドカーブチャートタイプでは、デフォルトで true です。

Default Value​デフォルト値​

false
