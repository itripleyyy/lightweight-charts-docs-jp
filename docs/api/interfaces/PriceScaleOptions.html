Version: 5.0

Interface: PriceScaleOptionsインターフェース: 価格スケールオプション

Structure that describes price scale options価格スケールオプションを説明する構造

Properties​プロパティ​

autoScale​

autoScale: boolean

Autoscaling is a feature that automatically adjusts a price scale to fit the visible range of data. Note that overlay price scales are always auto-scaled.オートスケーリングは、データの表示範囲に合わせて価格スケールを自動的に調整する機能です。 オーバーレイ価格スケールは常にオートスケーリングされることに注意してください。

Default Value​デフォルト値​

true

mode​

mode: PriceScaleMode

Price scale mode.価格スケールモード。

Default Value​デフォルト値​

{@link PriceScaleMode.Normal}

invertScale​

invertScale: boolean

Invert the price scale, so that a upwards trend is shown as a downwards trend and vice versa. Affects both the price scale and the data on the chart.価格スケールを反転させ、上昇トレンドを下降トレンドとして表示し、その逆も同様にします。 価格スケールとチャート上のデータの両方に影響します。

Default Value​デフォルト値​

false

alignLabels​

alignLabels: boolean

Align price scale labels to prevent them from overlapping.価格スケールのラベルを整列させ、重なりを防ぎます。

Default Value​デフォルト値​

true

scaleMargins​

scaleMargins: PriceScaleMargins

Price scale margins.価格スケールのマージン。

Default Value​デフォルト値​

{ bottom: 0.1, top: 0.2 }

Example​例​

chart.priceScale('right').applyOptions({
    scaleMargins: {
        top: 0.8,
        bottom: 0,
    },
});

borderVisible​

borderVisible: boolean

Set true to draw a border between the price scale and the chart area.価格スケールとチャートエリアの間に境界線を描画するにはtrueを設定します。

Default Value​デフォルト値​

true

borderColor​

borderColor: string

Price scale border color.価格スケールの境界線の色。

Default Value​デフォルト値​

'#2B2B43'

textColor?​

optional textColor: string

Price scale text color. If not provided LayoutOptions.textColor is used.価格スケールのテキストカラー。 指定されていない場合はLayoutOptions.textColorが使用されます。

Default Value​デフォルト値​

undefined

entireTextOnly​

entireTextOnly: boolean

Show top and bottom corner labels only if entire text is visible.全テキストが表示されている場合のみ、上部と下部のコーナーラベルを表示します。

Default Value​デフォルト値​

false

visible​

visible: boolean

Indicates if this price scale visible. Ignored by overlay price scales.この価格スケールが表示されているかどうかを示します。オーバーレイ価格スケールでは無視されます。

Default Value​デフォルト値​

true for the right price scale and false for the left. For the yield curve chart, the default is for the left scale to be visible.true は右の価格スケール用、false は左の価格スケール用です。 イールドカーブチャートでは、デフォルトで左のスケールが表示されます。

ticksVisible​

ticksVisible: boolean

Draw small horizontal line on price axis labels.価格軸ラベルに小さな水平線を描画します。

Default Value​デフォルト値​

false

minimumWidth​

minimumWidth: number

Define a minimum width for the price scale. Note: This value will be exceeded if the price scale needs more space to display it's contents.価格スケールの最小幅を定義します。 注意: この値は、価格スケールが内容を表示するためにより多くのスペースを必要とする場合に超えられます。

Setting a minimum width could be useful for ensuring that multiple charts positioned in a vertical stack each have an identical price scale width, or for plugins which require a bit more space within the price scale pane.最小幅を設定することで、垂直にスタックされた複数のチャートがそれぞれ同一の価格スケール幅を持つことを保証したり、価格スケールペイン内でより多くのスペースを必要とするプラグインに役立つことがあります。

Default Value​デフォルト値​

0

ensureEdgeTickMarksVisible​

ensureEdgeTickMarksVisible: boolean

Ensures that tick marks are always visible at the very top and bottom of the price scale, regardless of the data range. When enabled, a tick mark will be drawn at both edges of the scale, providing clear boundary indicators.データ範囲に関係なく、価格スケールの最上部と最下部にティックマークが常に表示されることを保証します。これを有効にすると、スケールの両端にティックマークが描画され、明確な境界インジケーターが提供されます。

Default Value​デフォルト値​

false
