Time scale時間スケール

info

This page is part of an iterative guide (where we build onto code from the previous steps).このページは反復ガイドの一部です（前のステップからコードを構築していきます）。

It is recommend that you follow the guide from the start. However, if you are only interested in the content on this page then take a look at the full code at the bottom of page for context of how this code fits into a working example.ガイドは最初から進めることをお勧めします。ただし、このページの内容にのみ興味がある場合は、ページの下部にある完全なコードを確認して、このコードが動作する例にどのように適合するかを理解してください。

In the previous step, we adjusted the vertical price scale, now let us adjust the horizontal time scale. We previously adjusted the border color of this scale and now we are going to adjust the starting 'zoom' level.前のステップでは、垂直価格スケールを調整しましたが、今度は水平時間スケールを調整しましょう。以前にこのスケールの境界色を調整しましたが、今度は開始の 'ズーム' レベルを調整します。

We will be adjusting the barSpacing option on the time scale which is used when the chart is first rendered to determine the horizontal 'zoom' level. The property sets 'The space between bars in pixels.', where a larger number will result in wider bars and fewer bars visible on the chart. The default value is 6 and we will be increasing it to 10 which will effectively 'zoom in' for the time scale.時間スケールの barSpacing オプションを調整します。これは、チャートが最初にレンダリングされるときに水平 'ズーム' レベルを決定するために使用されます。このプロパティは 'バー間のピクセル数' を設定し、大きな数値はバーを広くし、チャート上に表示されるバーの数を減少させます。デフォルト値は 6 で、これを 10 に増やすことで、時間スケールを効果的に 'ズームイン' します。

Adjusting settings for the time scale​時間スケールの設定を調整する​

We can get the ITimeScaleApi instance for the chart by evoking the timeScale() method on the chart reference.チャートの timeScale() メソッドを呼び出すことで、チャートの ITimeScaleApi インスタンスを取得できます。

Just as with the previous step, we can use the applyOptions() method on this API instance to adjust it's options. You can add the following code to the example at any point after the chart reference has been created, but we will place it directly below where we set the border color for the time scale. The options available are shown here: TimeScaleOptions.前のステップと同様に、このAPIインスタンスの applyOptions() メソッドを使用してオプションを調整できます。chart リファレンスが作成された後の任意のポイントに以下のコードを追加できますが、時間スケールの境界色を設定した直後に配置します。利用可能なオプションはここに示されています: TimeScaleOptions。

// Setting the border color for the horizontal axis
chart.timeScale().applyOptions({
    borderColor: '#71649C',
});

// Adjust the starting bar width (essentially the horizontal zoom)
chart.timeScale().applyOptions({
    barSpacing: 10,
});

The applyOptions() calls on the time scale were purposely split into two to show that it is possible to apply the options individually if that leads to cleaner code to read. It is also possible to apply both options in a single step as shown below:applyOptions() の呼び出しは、コードを読みやすくするためにオプションを個別に適用できることを示すために意図的に2つに分けられました。また、以下に示すように、両方のオプションを1つのステップで適用することも可能です。

// Example of applying both properties in a single call
chart.timeScale().applyOptions({
    borderColor: '#71649C',
    barSpacing: 10,
});

Auto fitting all the content​すべてのコンテンツを自動フィットさせる​

It is possible to auto fit all the content into the visable area of the chart by calling the fitContent() method on the time scale instance, for example: chart.timeScale().fitContent();時間スケールインスタンスで fitContent() メソッドを呼び出すことで、チャートの表示可能エリアにすべてのコンテンツを自動フィットさせることができます。例えば: chart.timeScale().fitContent();

      // Create the Lightweight Chart within the container element
      const chart = LightweightCharts.createChart(
        document.getElementById('container'),
        {
          layout: {
            background: { color: "#222" },
            textColor: "#C3BCDB",
          },
          grid: {
            vertLines: { color: "#444" },
            horzLines: { color: "#444" },
          },
        }
      );

      // Setting the border color for the vertical axis
      chart.priceScale().applyOptions({
        borderColor: "#71649C",
      });

      // Setting the border color for the horizontal axis
      chart.timeScale().applyOptions({
        borderColor: "#71649C",
      });

      // Adjust the starting bar width (essentially the horizontal zoom)
      chart.timeScale().applyOptions({
        barSpacing: 10,
      });

      // Get the current users primary locale
      const currentLocale = window.navigator.languages[0];
      // Create a number format using Intl.NumberFormat
      const myPriceFormatter = Intl.NumberFormat(currentLocale, {
        style: "currency",
        currency: "EUR", // Currency for data points
      }).format;

      // Apply the custom priceFormatter to the chart
      chart.applyOptions({
        localization: {
          priceFormatter: myPriceFormatter,
        },
      });

      // Generate sample data to use within a candlestick series
      const candleStickData = generateCandlestickData();

      // Create the Main Series (Candlesticks)
      const mainSeries = chart.addSeries(LightweightCharts.CandlestickSeries);
      // Set the data for the Main Series
      mainSeries.setData(candleStickData);

      // Changing the Candlestick colors
      mainSeries.applyOptions({
        wickUpColor: "rgb(54, 116, 217)",
        upColor: "rgb(54, 116, 217)",
        wickDownColor: "rgb(225, 50, 85)",
        downColor: "rgb(225, 50, 85)",
        borderVisible: false,
      });

      // Adjust the options for the priceScale of the mainSeries
      mainSeries.priceScale().applyOptions({
        autoScale: false, // disables auto scaling based on visible content
        scaleMargins: {
          top: 0.1,
          bottom: 0.2,
        },
      });

      // Adding a window resize event handler to resize the chart when
      // the window size changes.
      // Note: for more advanced examples (when the chart doesn't fill the entire window)
      // you may need to use ResizeObserver -> https://developer.mozilla.org/en-US/docs/Web/API/ResizeObserver
      window.addEventListener("resize", () => {
        chart.resize(window.innerWidth, window.innerHeight);
