Customization - Introductionカスタマイズ - はじめに

This tutorial provides an introduction to customizing Lightweight Charts™ appearance and functionality. It is not meant as an exhaustive tutorial but rather as a guided tour on how and where to apply options within the API to adjust specific parts of the chart. Along the way, we will provide links to the API documentation which outline the full set of options available for each part of the chart. It is highly recommended that you explore these API links to discover the wide range of possible customization and feature flags contained within Lightweight Charts™.このチュートリアルでは、Lightweight Charts™の外観と機能をカスタマイズする方法についての紹介を提供します。これは包括的なチュートリアルではなく、API内でオプションを適用してチャートの特定の部分を調整する方法と場所についてのガイドツアーです。その過程で、チャートの各部分に利用可能なオプションの完全なセットを概説したAPIドキュメントへのリンクを提供します。Lightweight Charts™に含まれる幅広いカスタマイズや機能フラグを発見するために、これらのAPIリンクを探索することを強くお勧めします。

Terminology​用語​

  • Data Series (aka data/dataset): A collection of data points representing a specific metric over time.データシリーズ（データ/データセットとも呼ばれる）： 特定のメトリックを時間の経過とともに表すデータポイントのコレクション。
  • Series Type: A series type specifies how to draw the data on the chart. For example, a line series type will plot the data series on the chart as a series of the data points connected by straight line segments. Available series types: Series types | Lightweight Chartsシリーズタイプ： シリーズタイプは、チャート上にデータを描画する方法を指定します。たとえば、ラインシリーズタイプは、データポイントを直線セグメントで接続した系列としてチャートにプロットします。利用可能なシリーズタイプ：シリーズタイプ | Lightweight Charts
  • Series: A combination of a specified series type and a data series.シリーズ： 指定されたシリーズタイプとデータシリーズの組み合わせ。
  • Price Scale: Price Scale (or price axis) is a vertical scale that mostly maps prices to coordinates and vice versa.価格スケール: 価格スケール（または価格軸）は、主に価格を座標にマッピングし、その逆も行う縦のスケールです。
  • Time Scale: Time scale (or time axis) is a horizontal scale at the bottom of the chart that displays the time of bars.時間スケール: 時間スケール（または時間軸）は、チャートの下部にある横のスケールで、バーの時間を表示します。
  • Crosshair: Thin vertical and horizontal lines centered on a data point in the chart.クロスヘア: チャート内のデータポイントに中心を合わせた細い縦線と横線です。

How to set up the example so you can follow along​例を設定して、進める方法​

This guide makes use of a single HTML file which you can download to your computer and run in the browser without the need for any build steps or web servers. The only thing required is an active internet connection such that the Lightweight Charts™ library can be downloaded from the CDN.このガイドでは、単一のHTMLファイルを使用しており、これをコンピュータにダウンロードして、ビルドステップやウェブサーバーなしでブラウザで実行できます。必要なのは、Lightweight Charts™ライブラリをCDNからダウンロードできるアクティブなインターネット接続だけです。

Provided below is the 'starting point' file for the guide which is a simple HTML page scaffolded out with a single div element (#container) and a JS function to generate the sample data set. At this point, you won't see anything on the page until we add the chart in the next step.以下に、ガイドの「出発点」ファイルを提供します。これは、単一のdiv要素（#container）とサンプルデータセットを生成するJS関数で構成されたシンプルなHTMLページです。この時点では、次のステップでチャートを追加するまで、ページには何も表示されません。

You can either:次のいずれかを行うことができます:

  • Download the file and then edit and run the example on your computer,ファイルをダウンロードしてから、コンピュータ上で例を編集して実行するか、
  • or open this JSFiddle and then edit and run the example within the browser.またはこのJSFiddleを開くして、ブラウザ内で例を編集して実行します。
tip

At the end of each section will be a link to download the example at that stage of the guide, and a full code block.各セクションの最後には、その段階の例をダウンロードするためのリンクと、完全なコードブロックがあります。
