Custom horizontal scaleカスタム水平スケール

The IHorzScaleBehavior interface allows you to customize the behavior of the horizontal scale. By default, this scale uses time values, but you can override it to use any other type of horizontal scale items, such as price values. The most typical use case is the creation of Options charts.IHorzScaleBehavior インターフェースを使用すると、水平スケールの動作をカスタマイズできます。デフォルトでは、このスケールは 時間 値を使用しますが、価格値などの他のタイプの水平スケールアイテムを使用するようにオーバーライドできます。最も典型的な使用例は、オプションチャートの作成です。

This guide will explain the IHorzScaleBehavior interface and how to implement it to create a horizontal scale using price values with customizable precision.このガイドでは、IHorzScaleBehavior インターフェースと、価格値を使用してカスタマイズ可能な精度で水平スケールを作成する方法を説明します。

Understanding the IHorzScaleBehavior interface​IHorzScaleBehavior インターフェースの理解​

The IHorzScaleBehavior interface consists of several methods that you need to implement to customize the horizontal scale behavior. Here's a breakdown of each method and its purpose:IHorzScaleBehavior インターフェースは、水平スケールの動作をカスタマイズするために実装する必要があるいくつかのメソッドで構成されています。各メソッドとその目的の内訳は次のとおりです：

options​

public options(): ChartOptionsImpl<HorzScaleItem>

This method returns the chart's current configuration options. These options include various settings that control the appearance and behavior of the chart. Implement this method to return the current options of your horizontal scale behavior.このメソッドは、チャートの現在の設定オプションを返します。これらのオプションには、チャートの外観と動作を制御するさまざまな設定が含まれています。このメソッドを実装して、水平スケールの動作の現在のオプションを返します。

setOptions​

public setOptions(options: ChartOptionsImpl<HorzScaleItem>): void

This method allows you to set or update the chart's configuration options. The provided options parameter will contain the settings you want to apply. Use this method to update the options when necessary.このメソッドは、チャートの設定オプションを設定または更新することを可能にします。提供されたoptionsパラメータには、適用したい設定が含まれます。このメソッドを使用して、必要に応じてオプションを更新します。

preprocessData​

public preprocessData(data: DataItem<HorzScaleItem> | DataItem<HorzScaleItem>[]): void

This method processes the series data before it is used by the chart. It receives an array of data items or a single data item. You can implement this method to preprocess or modify data as needed before it is rendered.このメソッドは、チャートで使用される前に系列データを処理します。データ項目の配列または単一のデータ項目を受け取ります。このメソッドを実装して、レンダリングされる前にデータを前処理または変更できます。

updateFormatter​

public updateFormatter(options: LocalizationOptions<HorzScaleItem>): void

This method updates the formatter used for displaying the horizontal scale items based on localization options. Implement this to set custom formatting settings, such as locale-specific date or number formats.このメソッドは、ローカリゼーションオプションに基づいて水平スケール項目を表示するために使用されるフォーマッタを更新します。ローカル特有の日付や数値フォーマットなど、カスタムフォーマット設定を行うために実装します。

createConverterToInternalObj​

public createConverterToInternalObj(data: SeriesDataItemTypeMap<HorzScaleItem>[SeriesType][]): HorzScaleItemConverterToInternalObj<HorzScaleItem>

This method creates and returns a function that converts series data items into internal horizontal scale items. Implementing this method is essential for transforming your custom data into the format required by the chart's internal mechanisms.このメソッドは、系列データ項目を内部の水平スケール項目に変換する関数を作成して返します。このメソッドを実装することは、カスタムデータをチャートの内部メカニズムが要求する形式に変換するために不可欠です。

key​

public key(internalItem: InternalHorzScaleItem | HorzScaleItem): InternalHorzScaleItemKey

This method returns a unique key for a given horizontal scale item. It's used internally by the chart to identify and manage items uniquely. Implement this method to provide a unique identifier for each item.このメソッドは、特定の水平スケール項目に対して一意のキーを返します。これは、チャートが項目を一意に識別し管理するために内部で使用されます。このメソッドを実装して、各項目に一意の識別子を提供します。

cacheKey​

public cacheKey(internalItem: InternalHorzScaleItem): number

This method returns a cache key for a given internal horizontal scale item. This key helps the chart to cache and retrieve items efficiently. Implement this method to return a numeric key for caching purposes.このメソッドは、指定された内部の水平スケールアイテムのためのキャッシュキーを返します。このキーは、チャートがアイテムを効率的にキャッシュおよび取得するのに役立ちます。このメソッドを実装して、キャッシング目的のために数値キーを返します。

convertHorzItemToInternal​

public convertHorzItemToInternal(item: HorzScaleItem): InternalHorzScaleItem

This method converts a horizontal scale item into an internal item that the chart can use. Implementing this method ensures that your custom data type is correctly transformed for internal use.このメソッドは、水平スケールアイテムをチャートが使用できる内部アイテムに変換します。このメソッドを実装することで、カスタムデータ型が内部使用のために正しく変換されることが保証されます。

formatHorzItem​

public formatHorzItem(item: InternalHorzScaleItem): string

This method formats a horizontal scale item into a display string. The returned string will be used for displaying the item on the chart. Implement this method to format your items in the desired way (e.g., with a specific number of decimal places).このメソッドは、水平スケールアイテムを表示用の文字列にフォーマットします。返される文字列は、チャート上でアイテムを表示するために使用されます。このメソッドを実装して、アイテムを希望の方法（例：特定の小数点以下の桁数で）でフォーマットします。

formatTickmark​

public formatTickmark(item: TickMark, localizationOptions: LocalizationOptions<HorzScaleItem>): string

This method formats a horizontal scale tick mark into a display string. The tick mark represents significant points on the horizontal scale. Implement this method to customize how tick marks are displayed.このメソッドは、水平スケールのティックマークを表示用の文字列にフォーマットします。ティックマークは、水平スケール上の重要なポイントを表します。このメソッドを実装して、ティックマークの表示方法をカスタマイズします。

maxTickMarkWeight​

public maxTickMarkWeight(marks: TimeMark[]): TickMarkWeightValue

This method determines the maximum weight for a set of tick marks, which influences their display prominence. Implement this method to specify the weight of the most significant tick mark.このメソッドは、ティックマークのセットの最大重量を決定し、それが表示の目立ち具合に影響を与えます。このメソッドを実装して、最も重要なティックマークの重量を指定します。

fillWeightsForPoints​

public fillWeightsForPoints(sortedTimePoints: readonly Mutable<TimeScalePoint>[], startIndex: number): void

This method assigns weights to the sorted time points. These weights influence the tick marks' visual prominence. Implement this method to provide a weighting system for your horizontal scale items.このメソッドは、ソートされた時間ポイントに重みを割り当てます。これらの重みは、目盛りの視覚的な重要性に影響を与えます。このメソッドを実装して、水平スケール項目のための重み付けシステムを提供してください。

Example​例​

Below is an example implementation of a custom horizontal scale behavior using price values. This example also includes customizable precision for formatting price values.以下は、価格値を使用したカスタム水平スケールの動作の実装例です。この例には、価格値のフォーマットのためのカスタマイズ可能な精度も含まれています。

Implement price-based horizontal scale​価格ベースの水平スケールを実装する​

  1. Define the custom localization options interfaceカスタムローカリゼーションオプションインターフェースを定義する

Extend the LocalizationOptions interface to include a precision property.LocalizationOptionsインターフェースを拡張して、precisionプロパティを含めます。

export interface CustomLocalizationOptions
    extends LocalizationOptions<HorzScalePriceItem> {
    precision: number;
}
  1. Define the type alias型エイリアスを定義する

Define a type alias for the horizontal scale item representing price values.価格値を表す水平スケール項目の型エイリアスを定義します。

export type HorzScalePriceItem = number;
  1. Implement the custom horizontal scale behavior classカスタム水平スケール動作クラスを実装する

The HorzScaleBehaviorPrice class implements the IHorzScaleBehavior interface, with additional logic to handle the precision provided in the custom localization options.HorzScaleBehaviorPriceクラスは、IHorzScaleBehaviorインターフェースを実装し、カスタムローカリゼーションオプションで提供された精度を処理するための追加のロジックを持っています。

function markWithGreaterWeight(a: TimeMark, b: TimeMark): TimeMark {
    return a.weight > b.weight ? a : b;
}

export class HorzScaleBehaviorPrice implements IHorzScaleBehavior<HorzScalePriceItem> {
    private _options!: ChartOptionsImpl<HorzScalePriceItem>;

    public options(): ChartOptionsImpl<HorzScalePriceItem> {
        return this._options;
    }

    public setOptions(options: ChartOptionsImpl<HorzScalePriceItem>): void {
        this._options = options;
    }

    public preprocessData(
        data: DataItem<HorzScalePriceItem> | DataItem<HorzScalePriceItem>[]
    ): void {
        // un-needed in this example because we do not require any additional
        // data processing for this scale.
        // The method is still required to be implemented in the class.
    }

    public updateFormatter(options: CustomLocalizationOptions): void {
        if (!this._options) {
            return;
        }
        this._options.localization = options;
    }

    public createConverterToInternalObj(
        data: SeriesDataItemTypeMap<HorzScalePriceItem>[SeriesType][]
    ): HorzScaleItemConverterToInternalObj<HorzScalePriceItem> {
        return (price: number) => price as unknown as InternalHorzScaleItem;
    }

    public key(
        internalItem: InternalHorzScaleItem | HorzScalePriceItem
    ): InternalHorzScaleItemKey {
        return internalItem as InternalHorzScaleItemKey;
    }

    public cacheKey(internalItem: InternalHorzScaleItem): number {
        return internalItem as unknown as number;
    }

    public convertHorzItemToInternal(
        item: HorzScalePriceItem
    ): InternalHorzScaleItem {
        return item as unknown as InternalHorzScaleItem;
    }

    public formatHorzItem(item: InternalHorzScaleItem): string {
        return (item as unknown as number).toFixed(this._precision());
    }

    public formatTickmark(
        item: TickMark,
        localizationOptions: LocalizationOptions<HorzScalePriceItem>
    ): string {
        return (item.time as unknown as number).toFixed(this._precision());
    }

    public maxTickMarkWeight(marks: TimeMark[]): TickMarkWeightValue {
        return marks.reduce(markWithGreaterWeight, marks[0]).weight;
    }

    public fillWeightsForPoints(
        sortedTimePoints: readonly Mutable<TimeScalePoint>[],
        startIndex: number
    ): void {
        const priceWeight = (price: number) => {
            if (price === Math.ceil(price / 100) * 100) {
                return 8;
            }
            if (price === Math.ceil(price / 50) * 50) {
                return 7;
            }
            if (price === Math.ceil(price / 25) * 25) {
                return 6;
            }
            if (price === Math.ceil(price / 10) * 10) {
                return 5;
            }
            if (price === Math.ceil(price / 5) * 5) {
                return 4;
            }
            if (price === Math.ceil(price)) {
                return 3;
            }
            if (price * 2 === Math.ceil(price * 2)) {
                return 1;
            }
            return 0;
        };
        for (let index = startIndex; index < sortedTimePoints.length; ++index) {
            sortedTimePoints[index].timeWeight = priceWeight(
                sortedTimePoints[index].time as unknown as number
            );
        }
    }

    private _precision(): number {
        return (this._options.localization as CustomLocalizationOptions).precision;
    }
}

This class provides additional precision control through localization options, allowing formatted price values to use a specific number of decimal places.このクラスは、ローカリゼーションオプションを通じて追加の精度制御を提供し、フォーマットされた価格値が特定の小数点以下の桁数を使用できるようにします。

Customize horizontal scale behavior​水平スケールの動作をカスタマイズする​

To use the custom horizontal scale behavior, instantiate the HorzScaleBehaviorPrice class and pass it to createChartEx.カスタム水平スケールの動作を使用するには、HorzScaleBehaviorPriceクラスをインスタンス化し、それをcreateChartExに渡します。

You can pass the custom option for precision within the localization property of the chart options.localizationプロパティ内でprecisionのカスタムオプションを渡すことができます。

const horzItemBehavior = new HorzScaleBehaviorPrice();
const chart = LightweightCharts.createChartEx(container, horzItemBehavior, {
    localization: {
        precision: 2, // custom option
    },
});
const s1 = chart.addSeries(LightweightCharts.LineSeries);
const data = [];
for (let i = 0; i < 5000; i++) {
    data.push({
        time: i * 0.25,
        value: Math.sin(i / 100),
    });
}
s1.setData(data);

Conclusion​結論​

The IHorzScaleBehavior interface provides a powerful way to customize the horizontal scale behavior in Lightweight Charts™. By implementing this interface, you can define how the horizontal scale should interpret and display custom data types, such as price values. The provided example demonstrates how to implement a horizontal scale with customizable precision, allowing for tailored display formats to fit your specific requirements.IHorzScaleBehaviorインターフェースは、Lightweight Charts™における水平スケールの動作をカスタマイズするための強力な方法を提供します。このインターフェースを実装することで、水平スケールが価格値などのカスタムデータタイプをどのように解釈し表示するかを定義できます。提供された例は、カスタマイズ可能な精度を持つ水平スケールを実装する方法を示しており、特定の要件に合わせた表示形式を可能にします。

  • and that there is an html div element on the page with an id of container.

// Lightweight Charts™ Example: Horizontal Price Scale
// https://tradingview.github.io/lightweight-charts/tutorials/how_to/horizontal-price-scale

function markWithGreaterWeight(a, b) {
    return a.weight > b.weight ? a : b;
}

/** @type {import('lightweight-charts').IHorzScaleBehavior} */
class HorzScaleBehaviorPrice {
    constructor() {
        this._options = {};
    }

    options() {
        return this._options;
    }

    setOptions(options) {
        this._options = options;
    }

    preprocessData(data) {}

    updateFormatter(options) {
        if (!this._options) {
            return;
        }
        this._options.localization = options;
    }

    createConverterToInternalObj(data) {
        return price => price;
    }

    key(internalItem) {
        return internalItem;
    }

    cacheKey(internalItem) {
        return internalItem;
    }

    convertHorzItemToInternal(item) {
        return item;
    }

    formatHorzItem(item) {
        return item.toFixed(this._precision());
    }

    formatTickmark(item, localizationOptions) {
        return item.time.toFixed(this._precision());
    }

    maxTickMarkWeight(marks) {
        return marks.reduce(markWithGreaterWeight, marks[0]).weight;
    }

    fillWeightsForPoints(sortedTimePoints, startIndex) {
        const priceWeight = price => {
            if (price === Math.ceil(price / 100) * 100) {
                return 8;
            }
            if (price === Math.ceil(price / 50) * 50) {
                return 7;
            }
            if (price === Math.ceil(price / 25) * 25) {
                return 6;
            }
            if (price === Math.ceil(price / 10) * 10) {
                return 5;
            }
            if (price === Math.ceil(price / 5) * 5) {
                return 4;
            }
            if (price === Math.ceil(price)) {
                return 3;
            }
            if (price * 2 === Math.ceil(price * 2)) {
                return 1;
            }
            return 0;
        };
        for (let index = startIndex; index < sortedTimePoints.length; ++index) {
            sortedTimePoints[index].timeWeight = priceWeight(
                sortedTimePoints[index].time
            );
        }
    }

    _precision() {
        return this._options.localization.precision;
    }
}

const horzItemBehavior = new HorzScaleBehaviorPrice();

const chartOptions = {
    layout: {
        textColor: 'black',
        background: { type: 'solid', color: 'white' },
    },
    localization: {
        precision: 2, // custom option
    },
};

/** @type {import('lightweight-charts').IChartApi} */
const chart = createChartEx(
    document.getElementById('container'),
    horzItemBehavior,
    chartOptions
);

const lineSeries = chart.addSeries(LineSeries, { color: '#2962FF' });

const data = [];
for (let i = 0; i < 5000; i++) {
    data.push({
        time: i * 0.25,
        value: Math.sin(i / 100) + i / 500,
    });
}

lineSeries.setData(data);

chart.timeScale().fitContent();
