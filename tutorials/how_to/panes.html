Panesパネル

Lightweight Charts™ allows you to create multiple panes in a single chart.Lightweight Charts™を使用すると、単一のチャートに複数のパネルを作成できます。

Using multiple panes in a charting library can be incredibly useful for a variety of analytical and visualization scenarios, especially when dealing with complex datasets or requiring detailed comparative analysis across different data dimensions.チャートライブラリで複数のパネルを使用することは、さまざまな分析や視覚化シナリオに非常に役立ちます。 特に、複雑なデータセットを扱う場合や、異なるデータ次元間で詳細な比較分析が必要な場合に有効です。

This example shows how to use panes in Lightweight Charts™. We will create a chart with two panes: the first one will display a stock's price over time and the second one will contain trading volume. The price and volume will be represented with candles and an area, respectively.この例では、Lightweight Charts™でパネルを使用する方法を示します。最初のパネルには株価の推移を表示し、2番目のパネルには取引量を表示するチャートを作成します。価格とボリュームはそれぞれキャンドルとエリアで表現されます。

You can see a full working example below.下に完全な動作例を見ることができます。

How to add a pane​パネルを追加する方法​

To introduce an additional pane into a chart, specify paneIndex during series creation.チャートに追加のパネルを導入するには、シリーズ作成時にpaneIndexを指定します。

Alternatively, you can invoke the moveToPane method on the series instance. Note that if the pane with specified index doesn't exist, it will be created.または、シリーズインスタンスでmoveToPaneメソッドを呼び出すことができます。指定したインデックスのパネルが存在しない場合は、新しく作成されます。

const volumeSeries = chart.addSeries(
    HistogramSeries,
    {
        priceFormat: {
            type: 'volume',
        },
    },
    1 // Pane index
);
// Moving the series to a different pane
volumeSeries.moveToPane(2);

If a series is moved out of a pane and no other series remain, the pane will be automatically removed.シリーズがパネルから移動され、他にシリーズが残っていない場合、そのパネルは自動的に削除されます。

Customizations​カスタマイズ​

Lightweight Charts™ provides options to customize the panes. You can adjust the pane separator color by specifying the separatorColor property in the layout.panes chart options, and use separatorHoverColor to change the separator color on hover.Lightweight Charts™ は、ペインをカスタマイズするオプションを提供します。layout.panes チャートオプションで separatorColor プロパティを指定することでペインのセパレーターの色を調整でき、separatorHoverColor を使用してホバー時のセパレーターの色を変更できます。

chart.applyOptions({
    layout: {
        panes: {
            separatorColor: '#ff0000',
            separatorHoverColor: '#00ff00',
            enableResize: false,
        },
    },
});

Lightweight Charts™ includes PaneApi that allows you to control each pane. The API has methods to get information on the pane such as getHeight(), paneIndex(), and getSeries(). It also contains methods to adjust the pane parameters, for example setHeight(height) and moveTo(paneIndex).Lightweight Charts™ には、各ペインを制御するための PaneApi が含まれています。このAPIには、getHeight()、paneIndex()、および getSeries() など、ペインに関する情報を取得するためのメソッドがあります。また、setHeight(height) や moveTo(paneIndex) など、ペインのパラメータを調整するためのメソッドも含まれています。

To get a PaneApi instance for each pane, you need to call the panes method on the ChartApi instance. Let's say we want to set the height of the second pane to 300px and move it to the first position.各ペインの PaneApi インスタンスを取得するには、ChartApi インスタンスの panes メソッドを呼び出す必要があります。例えば、2番目のペインの高さを300pxに設定し、それを最初の位置に移動したいとします。

const secondPane = chart.panes()[1];
secondPane.setHeight(300);
secondPane.moveTo(0);

Note that the minimum pane height is 30px. Any values lower than 30px will be ignored.最小ペイン高さは30pxであることに注意してください。30px未満の値は無視されます。

To remove the pane, you can call the removePane(paneIndex) method on the ChartApi instance.ペインを削除するには、ChartApi インスタンスの removePane(paneIndex) メソッドを呼び出すことができます。

chart.removePane(1);

Note that removing a pane also removes any series contained within it.ペインを削除すると、その中に含まれるすべてのシリーズも削除されることに注意してください。
